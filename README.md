Moodle "SEO" Theme Repository
===============================

An SEO theme for Moodle designed to optimize pages for search engines.

This is not a style theme — your original theme templates and styles are preserved. Simply select your current theme as the parent theme in the settings.

## How to Use and Activate

1. Go to **Site Administration → Appearance → Themes → SEO Settings**
2. Choose your current theme as the parent theme.
3. Select **SEO** as the active theme — no visual styles will be changed.

## Features

### Page Meta Tags

- **Keywords**
  - Generates keywords from tags present in the page context.
  - Allows overriding keywords or appending to existing ones.

- **Title**
  - Moodle already generates titles, but some plugin pages do not specify one — the page heading is used as a fallback.
  - Admins can modify the title of any page using SEO Manager.

- **Description**
  - Auto-generates descriptions for many page types:
    - **Home page:** Uses the site summary.
    - **Course public or enrollment page:** Uses the course summary.
    - **Public profiles:** Uses the user description.
    - **Blogs:** Uses the summary, or content as a fallback.
    - **Course category:** Uses the category description.
    - **Pages (generated by local_pg):** Uses part of the page content.
  - Can be manually overridden via SEO Manager to specify an exact description for any URL.

- **Robots**
  - Automatically adds a robots meta tag with instructions (`follow`, `index`).
  - Admins can override specific pages via SEO Manager to add `noindex`.
  - *(Not yet implemented: Advanced options for the robots meta tag.)*

- **Schema Markup**
  - Automatically generated for course pages.
  - Can be overridden for any page by advanced users via SEO Manager.
  - *(Not yet implemented: Automated schema generation for course categories, blogs, and public profiles.)*

### Page Live Analysis

- SEO score.
- Keyword density.
- Title and description length.
- Presence of headings.
- Keyword density in title, description, and content.
- Number of internal and external links on the page.
- Valid and broken link detection.
- Page preview as guest or crawler.
- Redirect detection and redirect URL.
- Page load time.
- Page download size.
- Whether the page is public and visible to crawlers.

### Sitemap Auto-Generation

- Automatically generates URLs for:
  - Home page.
  - Visible courses.
  - Visible categories.
  - Public blogs.
  - Public pages from local_pg.
- Any page can be marked to be included or excluded from the sitemap via SEO Manager.
- Admins can specify custom URLs to be added to the sitemap.

### robots.txt Auto-Generation

- Includes the URL of the generated sitemap (updated weekly by cron).
- Excludes the admin path.
